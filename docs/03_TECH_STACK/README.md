# 기술 스택 및 인프라

## 2.1 로컬 백엔드 시스템

- [ ] **로컬 API 서버**
  - RESTful API 또는 GraphQL (로컬호스트에서 실행)
  - 에디터 확장 프로그램과 통신
  - **로컬 동작**: 백그라운드 프로세스로 실행, 포트 바인딩

- [ ] **코드 분석 엔진**
  - Tree-sitter 또는 언어별 파서
  - 정적 분석 도구 통합 (ESLint, Pylint 등)
  - **로컬 동작**: 파일 시스템 직접 접근, 프로젝트 루트 기준 분석

- [ ] **AI/ML 모델 통합**
  - LLM API 연동 (OpenAI, Anthropic - 외부 API 호출)
  - 코드 임베딩 모델 (로컬 모델 또는 API)
  - **로컬 동작**: API 키 설정 → 네트워크 요청 → 결과 캐싱

## 2.2 프론트엔드 시스템

- [ ] **코드 에디터 통합**
  - VS Code Extension
  - JetBrains Plugin
  - 웹 기반 에디터

- [ ] **UI/UX**
  - 코드 하이라이팅
  - 제안 표시 인터페이스
  - 리뷰 결과 시각화
  - 실시간 피드백

## 2.3 로컬 데이터 관리

- [ ] **로컬 파일 시스템**
  - 프로젝트 디렉토리 직접 접근
  - Git 정보 읽기 (.git 디렉토리)
  - **로컬 동작**: 파일 시스템 API 활용, 상대 경로 기준 작업

- [ ] **로컬 캐싱 시스템**
  - 분석 결과 캐싱 (메모리 또는 디스크)
  - 파일 해시 기반 캐시 무효화
  - **로컬 동작**: `.cache` 디렉토리에 JSON 파일로 저장

- [ ] **로컬 설정 저장**
  - 사용자 설정 파일 (JSON/YAML)
  - 프로젝트별 설정 (.toolrc 파일)
  - **로컬 동작**: 홈 디렉토리 또는 프로젝트 루트에 설정 파일 저장
